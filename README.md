# Django Product Management Web Application

A full-stack web application built with Django to manage product data efficiently.  
The system allows users to create, view, update, and delete products with dynamic search functionality and a responsive user interface.

---

## Tech Stack

Backend  
- Python  
- Django  

Frontend  
- HTML  
- CSS  
- Bootstrap  
- JavaScript  

Database  
- SQLite  

Tools & Deployment  
- Git  
- GitHub  
- GitHub Actions (CI Workflow)  
- Render (Cloud Deployment)

---

## Features

- Product CRUD operations (Create, Read, Update, Delete)
- Product search functionality
- Backend validation using Django forms
- Database operations using Django ORM
- Responsive UI using Bootstrap
- CI workflow using GitHub Actions
- Automatic deployment on Render

---

## Architecture

This project follows Django's **MVT (Model – View – Template)** architecture.

Application flow:

User Request → URL → View → Model → Template → Response
